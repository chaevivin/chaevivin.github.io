---

 브라우저마다 가지고 있는 기본 설정이 달라서 똑같은 CSS 파일이라도 다 다르게 보일 수 있다. 그래서 CSS를 작성하기 전에 CSS를 초기화하는 것이 바람직하다. 

<br>

{% include codepen.html hash="eYVZaOJ" title="css_reset" %}

```html
<!DOCTYPE html>
<html lang="ko">
<head>
	<meta charset="UTF-8">
	<title>CSS Reset</title>
	<link rel="stylesheet" href="./css/main.css">
</head>
<body>
	<div class="box"></div>
</body>
</html>
```
```css
/* 초기화 */
body {
	margin: 0;
	padding: 0;
}
.box {
	width: 100PX;
	height: 100px;
	background: yellow;
}
```
